# OpenGate.Sample.DeviceFlow

Console app demonstrando `device authorization flow` com polling de token.

## Como usar

1. Habilite `EnableDeviceFlow` no servidor OpenGate.
2. Rode `samples/OpenGate.Sample.Basic`.
3. Execute: `dotnet run --project samples/OpenGate.Sample.DeviceFlow`.
4. Acesse a URL exibida no console e confirme o `user_code`.
